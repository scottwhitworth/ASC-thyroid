About This Demonstration

In 2010, The Bethesda System for Reporting Thyroid Cytopathology atlas was released. The atlas is intended to supplement the synopsis (insert link) published following the National Cancer Institute Thyroid Fine Needle Aspiration State of the Science Conference. This demonstration application is intended to function as a prototype example of how a laboratory can provide a central, accessible explanation of its changing thyroid reports to the providers that it services as it adopts the recommended terminology.

Open Source

This application was developed using the Python programming language against the Django web development framework. The source code is BSD-licensed and is available on Github (insert link). I accept pull requests and patches if you would like to contribute.

Disclaimer

This web application is intended to serve as an educational demonstration and should not be used to direct patient care.
